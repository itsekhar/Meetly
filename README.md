# Meetly

Meetly is a versatile video calling and meeting application built with modern web technologies. 

## Tech Stack
- **Framework:** Next.js (React)
- **Database:** MongoDB (via Mongoose)
- **Authentication:** JWT & bcryptjs
- **Video Calling SDK:** ZegoCloud UIKits
- **Styling:** Tailwind CSS

## Features
- Secure user authentication and authorization
- High-quality video calls and real-time communication
- Intuitive and responsive user interface

## Setup Instructions

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Configure Environment Variables:**
   Create a `.env` file in the root directory and add the necessary configuration parameters (e.g., MongoDB URI, ZegoCloud credentials, JWT Secrets).

3. **Start the Development Server:**
   ```bash
   npm run dev
   ```

4. **Access the Application:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.
